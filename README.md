This code generates [bifurcation](https://en.wikipedia.org/wiki/Bifurcation_diagram) and [cobweb](https://en.wikipedia.org/wiki/Cobweb_plot) diagrams, used in the study of [iterated functions](https://en.wikipedia.org/wiki/Iterated_function).

The `.nb` files can only be opened with Mathematica; see PDFs.
